# DEARTEENLINEA

Dearteenlinea es una galeria web donde se muestran obras de artistas consolidados, artistas jovenes y arte de mercado secundario. Todos estos son curados por dearteenlinea para ofrecer obras de calidad a sus visitantes.

# qullqa

Qullqa es una plataforma web que se le vende a artistas y a coleccionistas para que gestionen las obras que crean y que coleccionan proporcionando funcionalidades que otorgan valor. Los artistas que tienen qullqa pueden publicar sus obras para ser visibles al público líbremente en un espacio que se llama qullqa gallery.
